# CLAUDE.md — aiops repo conventions

## File size limit

**Keep every source file under 500 lines.** When a file approaches the limit, split it before it crosses it.

## Language

- UI strings: Chinese by default, English as second language in i18n
- Code comments: English
- Commit messages: English

## Target project conventions

For conventions that apply to codebases using aiops (TypeScript/React splitting order, etc.), see `docs/target-project-conventions.md`.
